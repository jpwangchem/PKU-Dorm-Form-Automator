# 🏢 北京大学宿舍入住凭单自动化填充工具

这是一个基于 Python 的自动化工具，用于将 Excel 中的学生住宿信息批量填充到 Word 模板中并生成凭单。该工具专为处理北京大学宿舍入住流程的行政人员设计，旨在提高发放凭单的效率。

---

## 🌟 核心功能

* **自动双生配对**：自动将两名学生的信息组合到同一个 Word 文档中（一页 A4 纸包含两个表格），最大程度节省纸张。
* **强大的正则清洗**：
* **楼栋**：自动提取数字及后缀（例如：“燕园1楼” -> “1楼”）。


* **房间/床位**：自动剔除汉字等非数字字符（例如：“3号床” -> “3”），使凭单排版更简洁。


* **精准关键词匹配**：专门针对模板中的 `姓  名` 和 `楼  号` 进行优化（匹配中文表单中常见的双空格间距）。
* **智能命名规则**：生成的文件以 `学工号-姓名&学工号-姓名.docx` 格式命名，极大地方便了文件的检索与分发。

## 🚀 快速上手

### 1. 环境准备

确保你的电脑已安装 Python 3.12+，并安装以下依赖库：

```bash
pip install pandas python-docx openpyxl

```

### 2. 配置说明

* **数据文件 (`data.xlsx`)**：必须包含以下列：`学工号`、`姓名`、`楼栋`、`房间`、`床位`。
* **Word 模板 (`doc.docx`)**：运行代码前，请先手动填好模板中的固定信息（如 `院系`、`年级`、`经办人`、`时间` 等）。

### 3. 执行脚本

在终端或命令行运行：

```bash
python main.ipynb

```

## 📊 数据映射与处理逻辑

工具并非简单的复制粘贴，它会根据以下逻辑对数据进行“清洗”：

| 来源 (Excel) | 目标 (Word) | 正则/处理逻辑 | 效果示例 |
| --- | --- | --- | --- |
| **姓名** | `姓  名` | 去除两端空格 | " 张三 " → "张三" |
| **楼栋** | `楼  号` | `(\d+[^\d\s]?)` | "燕园1楼" → "1楼" |
| **房间** | `房间号` | `\d+` | "102室" → "102" |
| **床位** | `床位号` | `\d+` | "3号床" → "3" |

## 📂 项目结构

```text
.

├── data.xlsx            # 学生数据源（请妥善保管，勿上传至公开仓库！）

```

## ⚙️ 技术细节说明

* **配对逻辑**：利用 Python 的列表切片 `[df.iloc[i:i + 2]]` 以步长为 2 遍历 Excel 记录。
* **字体控制**：程序强制将填入的文字设置为 **10.5磅**（对应中文字号中的五号字）并 **左对齐**，确保无论 Excel 原格式如何，生成的凭单都整齐美观。
* **数据校验**：自动检查必填字段是否为 `NaN` 或空字符串。若关键信息缺失，将跳过该组以避免生成半残的凭单。
